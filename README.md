# Go tools

## 开始
Go tools [Go](https://go.dev/) 版本为 [1.19](https://go.dev/doc/devel/release#go1.19.0). 

直接在代码中导入:
```sh
import "github.com/revevide/gtools"
```
或者:
```sh
go get -u github.com/revevide/gtools
```
