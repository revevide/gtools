## Getting started
Gin requires [Go](https://go.dev/) version [1.19](https://go.dev/doc/devel/release#go1.19.0) or above.
Import in your code:
```sh
import "github.com/revevide/gtools"
```
Alternatively, use go get:
```sh
go get -u github.com/revevide/gtools
```
